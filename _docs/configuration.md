Jekyllではサイトをどのように構築するか、柔軟にカスタマイズすることができます。これらのオプションは、サイトのrootディレクトリにある`_config.yml`か`_config.toml`で指定することができます。また、ターミナルで`jekyll`を実行するときにflagsとして設定することもできます。

executable in the terminal. -->
